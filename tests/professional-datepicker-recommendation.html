📦 نصب و استفاده از Persian DatePicker (توصیه شده)

روش 1: CDN

<!-- CSS --> <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/persian-datepicker@1.2.0/dist/css/persian-datepicker.min.css"> <!-- JavaScript --> <script src="https://code.jquery.com/jquery-3.6.0.min.js"></script> <script src="https://cdn.jsdelivr.net/npm/persian-datepicker@1.2.0/dist/js/persian-datepicker.min.js"></script>

روش 2: NPM

npm install persian-datepicker

استفاده:

$('#date-input').persianDatepicker({ format: 'YYYY/MM/DD', initialValue: false, autoClose: true, calendar: { persian: { locale: 'fa' } }, navigator: { enabled: true }, toolbox: { enabled: true } });
